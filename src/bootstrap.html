Bootstrap

Bootstrap är ett av världens mest använda CSS-ramverk och utvecklas ursprungligen av Twitter. Det erbjuder ett omfattande bibliotek av färdiga komponenter och en mängd hjälpsamma klasser för att snabbt bygga moderna webbplatser.

Besök Bootstraps webbplats

Utvärdering av Bootstrap - ramverk

Fördelar

Bootstrap erbjuder färdiga komponenter som knappar, tabeller och dropdown-meny, vilket bidrar till snabbare utveckling och sparar mycket tid. Bootstrap underlättar att bygga mobilanpassade layouter.

Nackdelar

Å andra sidan har Bootstrap några nackdelar såsom att många webbplatser som använder Bootstrap ser likadana ut om man inte skriver extra CSS för att anpassa stilen och bryta sig loss från standardutseendet, dessutom Bootstrap är byggt med SCSS som kräver SCSS-kompilator för att kunna tolka och omvandla SCSS-kod till vanlig CSS.

Mina upplevelser

Att arbeta med Tailwind CSS var väldigt smidigt. Det gav mig möjlighet att bygga snabbt responsiv layout och använda färdiga komponenter för tabeller, formulär och navigeringsmeny. Dock noterade jag att det krävdes lite extra CSS för att få ett mer unikt utseende och även SCSS-kompilator för att omvandla SCSS-kod till vanlig css, därför jag installerade Parcel, npm-paket.
