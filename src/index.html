Startsida

Denna webbplats är skapad som en del av uppgift där målet är att laborera, utvärdera och använda två olika CSS-ramverk för att bygga sammanhängande webbplats med flera undersidor.

Syftet med uppgiften

Uppgift går ut på att:

  • Skapa en webbplats med minst fyra sidor
  • Använda två olika CSS-ramverk, ett för varje undersida
  • Dra nytta av ramrerkens funktioner för att styla tabeller, formulär, knappar och layout.
  • Säkerställa att webbplatsen fungerar bra både stora och små skärmar.
  • Implementera navigering mellan sidorna, även om den ser olika ut beroende på ramverk.

Valda CSS-ramverk

För denna uppgift har jag valt att arbeta med följande ramverk:

  • Bootstrap - Ett populärt och kraftfullt ramverk som erbjuder färdiga komponenter och ett rutnätsberat system för responsiv design.
  • Tailwind CSS - Ett utility-first ramverk som ger stor flexibiltet genom att använda klasser direkt i HTML för styling.
